Ai Startup Momentum Radar

A simple VC-style dashboard that tracks ai startup momentum and visualizes them with interactive charts.

What it shows
- Momentum bubble chart
- Momentum trend over time
- Leaderboard
- Sector treemap

Tech
- Python
- Streamlit
- Pandas
- Plotly

streamlit run app.py
